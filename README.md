# Project: CONQUERER BLOG APP
Welcome to Conquerer Blog App REST API documentation.

First of all, all the endpoints return a JSON datatype response and require cookies in order to send Json Web Tokens easily, automatically and securely thanks to httpOnly option. There are only 2 endpoints which don't require JWT and those are Register and Login endpoints. All the request data have to send via Body ( urlencoded ) expect JWT. JWTs're automatic thanks to cookies by default.

Let's check our API.

**Known Error Codes**

- \-1 - Login is required
    
- 0 - This email is already exist
    
- 1 - Inapproprate request
    
- 2 - Password length must be at least 8 characters
    
- 3 - Invalid login information
    
- 5 - Session is expired. Please login again.
    
- 6 - Unexpedted error is occured
    
- 8 - Invalid post id
    
- 9 - Missing information
    
- 12 - Something went wrong check your data and try again
